    NOTIFICACIONES

    Este código crea notificaciones visuales en una página web para mostrar mensajes importantes, como éxitos o errores. Puedes personalizar el tipo de notificación, su posición en la pantalla y el tiempo que permanece visible. Además, hay una función sencilla para mostrar notificaciones básicas.

    Ver La Documentacion

    MODAL

    Modal Básico: Crea un modal simple que se activa al hacer clic en botones y se cierra con un botón de cierre. Proporciona una manera fácil de mostrar información adicional.

    Modal Complejo: Define una función para crear un modal más detallado con título, texto, imagen y hasta tres botones. Ofrece opciones adicionales para personalizar y mostrar contenido detallado de manera interactiva.
